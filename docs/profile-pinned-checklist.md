# GitHub Profile 置顶配置清单

这份清单用于把 GitRadar 配成一个适合放在 GitHub Profile pinned repositories 区域的项目卡片。

目标不是“仓库能被看到”，而是让别人一眼看懂：

- 这是个什么产品
- 它和普通热榜工具有什么区别
- 当前版本已经做到什么程度
- 点进去后应该先看哪里

## 推荐使用的说明文案

### 置顶仓库标题下的一句话说明

推荐优先使用这版：

```text
一个可解释的 GitHub 开源兴趣雷达，不只告诉你今天什么项目热，还会逐渐学会你真正关心什么。
```

更偏工程能力的一版：

```text
一个带证据化中文日报、反馈闭环、环境指纹和可复盘归档的 GitHub 开源项目发现雷达。
```
