## 组合使用建议

如果你要在 GitHub 和社交平台同步发布 GitRadar，推荐顺序：

1. GitHub About 使用中文版短描述
2. Profile pinned repo 或仓库推广使用置顶预览图
3. Release 页面或发版动态使用 release 封面图
4. 想强调“这不是概念图”时，再补企业微信实发样例图
